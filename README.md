# Página web Corporación Factor C

Sitio web estático, editable y responsive para Corporación Factor C.

## Estructura

- `index.html`: página principal.
- `portafolio.html`: portafolio de servicios de CFC.
- `odesi.html`: marca ODESI / Observatorio de Densidad Sindical con tablero Power BI.
- `sindicatos.html`: acerca de los sindicatos, logos y directorio/buscador.
- `incidencia.html`: galería de incidencia sindical.
- `contacto.html`: datos de contacto y formulario a WhatsApp.
- `styles.css`: estilos generales.
- `script.js`: menú móvil, acordeón, buscador de sindicatos y formulario WhatsApp.
- `assets/`: logos, fotos, ODESI y PDF del portafolio.
